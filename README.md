# 刘阳短视频作品集

这是刘阳的个人短视频作品集网站，用于展示科研内容传播、实验室场景短视频和内容策略案例。

## GitHub Pages

推荐仓库名：

```text
liuyang-portfolio
```

发布后网址通常是：

```text
https://Lexie1997.github.io/liuyang-portfolio/
```

如果仓库名使用 `Lexie1997.github.io`，网址则是：

```text
https://Lexie1997.github.io/
```

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 更新视频

视频卡片在 `script.js` 的 `videos` 数组中维护。微信视频号、小红书通常不适合 iframe 嵌入，所以当前采用封面图 + 外链跳转的方式。
